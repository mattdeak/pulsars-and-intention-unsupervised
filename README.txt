Getting the Code
----------------

Simply clone the following repo:

https://github.com/mattdeak/pulsars-and-intention-unsupervised

It contains all necessary data.


Setting up the Environment
-----------------
In a python 3.7 environment, simply run `pip install requirements.txt` from the top-level directory.

Running the Code
----------------

To run all experiments, simply run

`python experiments.py`

To generate the plots, simply run

`python make_plots.py`

This must be done after running the experiments.
